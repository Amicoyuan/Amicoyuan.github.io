L2-013 红色警报 (25 分)

L2-013 红色警报 (25 分)

战争中保持各个城市间的连通性非常重要。本题要求你编写一个报警程序，当失去一个城市导致国家被分裂为多个无法连通的区域时，就发出红色警报。注意：若该国本来就不完全连通，是分裂的k个区域，而失去一个城市并不改变其他城市之间的连通性，则不要发出警报。

输入格式：

输入在第一行给出两个整数N（0 < N ≤ 500）和M（≤ 5000），分别为城市个数（于是默认城市从0到N-1编号）和连接两城市的通路条数。随后M行，每行给出一条通路所连接的两个城市的编号，其间以1个空格分隔。在城市信息之后给出被攻占的信息，即一个正整数K和随后的K个被攻占的城市的编号。

注意：输入保证给出的被攻占的城市编号都是合法的且无重复，但并不保证给出的通路没有重复。

输出格式：

对每个被攻占的城市，如果它会改变整个国家的连通性，则输出Red Alert: City k is lost!，其中k是该城市的编号；否则只输出City k is lost.即可。如果该国失去了最后一个城市，则增加一行输出Game Over.。

输入样例：

5 4
0 1
1 3
3 0
0 4
5
1 2 0 4 3

输出样例：

City 1 is lost.
City 2 is lost.
Red Alert: City 0 is lost!
City 4 is lost.
City 3 is lost.
Game Over.

程序源代码：

邻接矩阵建图 + DFS统计连通分量数

对于无向图来说，dfstrave()中dfs()或bfstrave()中bfs()的调用次数等于该图的连通分量数

无向图中的极大连通子图称为连通分量

极大连通子图必须连通，且包含尽可能多的顶点和边

#include<bits/stdc++.h>
using namespace std;
int n,m;
int g[505][505];						//邻接矩阵存图
int visit[505];							//标记访问数组
int lost[505];							//记录被攻占的城市
void dfs(int node){						//正常的DFS
    visit[node]=1;
    for(int i=0;i<n;i++)
    {
        if(visit[i]==0&&g[node][i]==1)
        {
            dfs(i);
        }
    }
}
int dfstrave(){
    int cnt=0;
    memset(visit,0,sizeof(visit));			//初始化访问数组
    for(int i=0;i<n;i++){
        if(lost[i]==1) visit[i]=1;			//如果该城市被攻占，则标记他为以访问
    }
    for(int i=0;i<n;i++){
        if(visit[i]==0){
            dfs(i);
            cnt++;      					//统计连通分量数
        }
    }
    return cnt;
}
int main()
{
    cin>>n>>m;
    memset(lost,0,sizeof(lost));			//初始化
    memset(g,0,sizeof(g));					//初始化
    for(int i=0;i<m;i++){
        int a,b;
        cin>>a>>b;
        g[a][b]=g[b][a]=1;					//存储无向图
    }
    int cnt=dfstrave();						//获得初始的连通分量数
    int k;
    cin>>k;
    for(int w=0;w<k;w++){
        int city;
        cin>>city;
        lost[city]=1;						//记录该城市被攻占
        for(int i=0;i<n;i++){
            if(g[city][i]==1){				
                g[city][i]=g[i][city]=0;	//标记和他相连的城市不再相连
            }
        }
        int tempcnt=dfstrave();				//当前的连通分量
        if(tempcnt<=cnt){					//如果当前的连通分量小于等于原来的连通分量，说明图的连通性没有被改变
            printf("City %d is lost.\n", city);
        }else{								//否则，连通性被改变了
            printf("Red Alert: City %d is lost!\n", city);
        }
        cnt=tempcnt;						//更新连通分量
        if(w==n-1){							//城市已经被攻占完毕
            printf("Game Over.\n");
        }
    }
    return 0;
}
